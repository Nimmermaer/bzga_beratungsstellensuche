Sitemap
-------
Sollte die Extension **seo** in Ihrer Installation verwendet werden, dann können Sie eine **spezielle Sitemap** für die Beratungsstellen generieren lassen.

Der **Aufruf der Sitemap** könnte dabei folgendermaßen aussehen:
http://www.ihre-domain.com/?type=1533906435

.. hint:: Die Parameter SINGLE_PID und STORAGE_FOLDER_PID müssen bitte durch die korrekten Angaben ersetzt werden.
