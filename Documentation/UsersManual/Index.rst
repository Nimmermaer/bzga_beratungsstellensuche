Redakteur
=========

.. only:: html

	Dieses Kapitel beschreibt die Verwendung der Erweiterung aus Sicht eines Redakteurs.

	Der Bereich "*Der Anfang*" zeigt auf, wie Datensätze angelegt bzw. importiert werden und wie die Datensätze im Frontend über die Einbindung der "*Plugins*" dargestellt werden.
	Im Bereich "*Datensätze*" werden die einzelnen Datensätze mit Ihren dazugehörigen Feldern im Detail beschrieben.
